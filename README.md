# 📱 Pix Apps Project

This is a React-based App Store website where users can explore a wide variety of apps categorized by type. The project uses **React Router** for navigation, **Firebase** for backend services (authentication & database), and a **slider library** for showcasing featured apps.

## 🚀 Features

- Browse apps by category
- View app details with descriptions
- website have 3 nav item when click nav item re direct this page
- User authentication (login/signup)
- Featured apps slider on the homepage
- Responsive design for mobile and desktop

## 🛠️ Tech Stack

- **Frontend:** React, React Router DOM
- **Backend:** Firebase (Firestore, Auth)
- **Slider Library:** Swiper.js
- **Styling:** CSS / TailwindCSS /daisyUI
